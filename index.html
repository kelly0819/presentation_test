Agent 3 — 개인화 추천 (Hybrid Architecture)

RECOMMENDATION ENGINE
① 협업 필터링

사용자-상품 행동 매트릭스 기반. 유사 사용자가 선호한 상품 후보 Top-N 추출.
click 0.1 / dwell 0.2 / cart 0.5 / purchase 1.0

② 콘텐츠 기반 필터링

사용자 벡터 ↔ 상품 벡터 간 코사인 유사도 계산. 개인화 추천의 핵심 로직.

③ 구매 사유 RAG

구매 사유 텍스트 임베딩 → Vector DB 저장 → 유사 사유 검색 → LLM이 feature 벡터 변환하여 선호 보정.

최종 사용자 벡터 공식
# 암묵적(행동) × 0.4 + 명시적(RAG) × 0.6
final_vec[f] = existing[f] * 0.4 + rag[f] * 0.6

# 예시
long_lasting: 0.7×0.4 + 0.9×0.6 = 0.82
drying: 0.3×0.4 + 0.1×0.6 = 0.18 ↓ 건조함 회피 강화
Cold Start 시: 협업 필터링 생략 → RAG + 콘텐츠 기반으로 보완
RAG 결합 이유
이유설명
암묵+명시 결합무엇을 선택했는지 + 왜 선택했는지 동시 반영
정확도 향상건조함·트러블 등 회피 feature 정밀 반영
Cold Start 완화소량 텍스트만으로 초기 추천 품질 보완
설명 가능성"지속력이 중요한 사용자" 등 추천 근거 제시
